# Basic In-Memory Bank System

This is a simple bank system implemented in Python that stores account data in memory. It supports:
- Account creation
- Deposit and withdrawal
- Balance inquiry

## Usage
Run the script to see a demo of the bank system:

```bash
python3 bank_system.py
```

You can modify the `__main__` section in `bank_system.py` to test more features.
